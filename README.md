

# 📊 Enterprise Dashboard JS

Dashboard moderno com gráficos animados e estatísticas empresariais fictícias.

Projeto desenvolvido com foco em visual profissional e organização de código, utilizando apenas tecnologias front-end.

---

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Chart.js

---

## 📌 Funcionalidades

* 📊 Gráficos animados
* 📈 Estatísticas empresariais
* 🎨 Visual moderno estilo sistema corporativo
* ⚡ Interface rápida e responsiva
* 📱 Layout adaptável
